# 프로젝트 요약

## 목적

이 프로젝트는 Olist 브라질 이커머스 데이터를 분석해 홈앤리빙 성장 전략을 만드는 것을 목표로 합니다. 분석은 셀러 행동, SKU 폭, 카테고리 성과, 고객 구매 빈도, 리텐션 가능성에 초점을 둡니다.

## 검토한 현재 원본 파일

마크다운 파일:

- `README.md`: 최소한의 프로젝트 제목만 포함합니다.
- `README2.md`: 홈앤리빙 성장 전략 서술 문서입니다.
- `README3.md`: `README2.md`와 같거나 거의 같은 문서입니다.

노트북 파일:

- `preprocess_all.ipynb`: Olist 원본 CSV를 불러오고 정제 테이블을 준비합니다.
- `olist_home_living_eda.ipynb`: 탐색 분석, 카테고리 성과, 시각화, 전처리 로직을 포함합니다.
- `olist_home_living_eda copy.ipynb`: EDA 노트북의 복사본 형태입니다.
- `olist_home_living_project.ipynb`: 상대 경로 로직이 포함된 홈앤리빙 전략 보고서 노트북입니다.
- `olist_home_living_project2.ipynb`: 프로젝트 노트북과 같거나 거의 같은 노트북입니다.
- `brazil_home_decor_strategy.ipynb`: 최종 전략 서술과 홈데코 카테고리 분석을 포함합니다.

## 분석 흐름

1. Olist CSV 파일을 불러옵니다.
2. 리뷰 중복, 비정상 주문 날짜 순서, 누락 상품 카테고리, 잘못된 상품 무게, 결제 이상치, 위치 중복을 정리합니다.
3. 주문, 고객, 상품, 셀러, 결제 데이터를 병합합니다.
4. 필요한 경우 상품 카테고리명을 번역 테이블과 연결합니다.
5. 매출, 주문 수, 고객 수, 반복 구매율, 셀러 집중도, SKU 수를 기준으로 홈앤리빙 카테고리 성과를 평가합니다.
6. 분석 결과를 셀러와 고객 성장 추천안으로 전환합니다.

## 핵심 비즈니스 메모

- 셀러 매출은 상위 셀러에게 집중되어 있습니다.
- SKU 폭은 셀러 성장 레버로 다룹니다.
- 전략 메모에서는 18개 SKU를 실무적 목표 기준으로 제시합니다.
- 3회 이상 구매한 고객은 리텐션 가능성이 더 높습니다.
- 추천 프로그램은 셀러 SKU 확장과 고객 리텐션 프로모션을 함께 추진하는 방향입니다.

## 깃허브 정리 기준

- 기존 루트 파일은 수정하지 않는 것을 원칙으로 했습니다.
- 깃허브에서 보기 쉬운 정리 노트북을 `notebooks/`에 새로 만들었습니다.
- 새 문서는 `docs/`와 `reports/`에 만들었습니다.
- 데이터 불러오기 예시는 `notebooks/` 기준 상대 경로를 사용합니다.
